# 💳 Credit Card Financial Dashboard

### 📌 Project Overview

This project demonstrates an end-to-end Power BI solution built to analyze credit card transactions and customer behavior.

The goal was to convert raw transactional data into a dynamic, insight-driven dashboard that helps businesses monitor financial performance, identify spending trends, and optimize decision-making.

This project covers the complete analytics workflow — from data extraction to visualization and business insights.

### 📊 Key Business Metrics

- 💰 Total Revenue / Transaction Amount
- 🧾 Total Transactions Count
- 👥 Total Customers
- 💳 Average Spend per Customer
- 📈 Month-over-Month Growth

### 🔍 Key Insights

- 📅 Clear monthly and seasonal spending trends identified
- 💳 Certain card categories contribute significantly higher revenue
- 👥 High-value customers drive a major portion of revenue
- 📍 Spending behavior varies across regions and demographics
- ⚠️ Opportunity to improve low-performing segments


### 🧠 Business Questions Answered

- Which customer segments generate the highest revenue?
- What are the spending trends over time?
- Which card types are most profitable?
- How does customer behavior vary across regions?
- What is the average transaction value and growth trend?
